# ZeppOS Monorepo Overlay

本目录保存当前宿主仓库需要的全局索引和 overlay 路由。它不是单项目文档，也不是官方 ZeppOS API 文档；目的是让 AI 在不扫描全仓的情况下定位项目、RE 证据链和历史迁移输入。

| 文件 | 用途 |
|------|------|
| `project-index.md` | 跨项目查找时使用的宿主项目索引 |
| `knowledge-map.yaml` | 宿主 overlay 路由：项目文档、RE 原始证据链、历史 memory |

单项目决策仍写入目标项目 `PROJECT.md`；可复用 ZeppOS 结论写入 `self-knowledge/`。
