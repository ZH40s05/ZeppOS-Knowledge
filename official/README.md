# 官方资料

本目录保存 ZeppOS 官方资料的版本快照。内容来自 Zepp Health 官方仓库或官方文档推荐列表。

## 快照内容

| 路径 | 来源 | 用途 |
|------|------|------|
| `docs/zeppos-docs/` | `zepp-health/zeppos-docs` | 官方文档、API、设计规范、工具说明 |
| `samples/zeppos-samples/` | `zepp-health/zeppos-samples` | 官方小程序、表盘、运动扩展样例 |
| `libraries/zml/` | `zepp-health/zml` | 官方推荐通信/网络工具库 |

## 维护规则

- 不在快照内直接修正文档或源码。
- 需要解释、纠错或补充时，写入 `../self-knowledge/`。
- 每次更新快照必须记录来源 commit、日期和许可证。
