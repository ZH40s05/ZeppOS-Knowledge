# 分类索引

本目录为人类和 AI 提供中文路由入口。需要查资料时先读这里，再进入对应快照或自有知识文件。

| 索引 | 主题 |
|------|------|
| `routing.yaml` | AI/脚本使用的 canonical 领域知识路由表 |
| `api.md` | 官方 API、`app.json`、路由、设备列表 |
| `ui.md` | UI 控件、设计规范、真机 UI quirks |
| `communication.md` | App/Side Service 通信、网络请求、ZML |
| `storage.md` | 文件、存储、EasyStorage、SysProp 摘要 |
| `ble.md` | BLE、ZML、easy-ble、HID 结论 |
| `re.md` | RE 开发摘要、隐藏 API、WiFi 可行性 |
| `samples.md` | 官方样例分类入口 |
| `libraries.md` | 官方推荐库、社区库、自维护库 |
| `../ai-workflow/README.md` | AI 协作 workflow、skill/tool 和验证入口 |

结论类型标记：

- `official`: 官方文档或官方样例确认。
- `true-device confirmed`: 真机验证确认。
- `RE confirmed`: 逆向工程证据确认。
- `inferred`: 基于多来源推断，仍需谨慎。
- `needs test`: 需要补充真机或构建验证。
