# 官方样例索引

官方样例快照位于 `official/samples/zeppos-samples/`。

| 类型 | 路径 |
|------|------|
| 小程序样例 | `official/samples/zeppos-samples/application/` |
| 表盘样例 | `official/samples/zeppos-samples/watchface/` |
| 运动扩展样例 | `official/samples/zeppos-samples/workout-extensions/` |

使用样例时，优先读取具体项目的 `app.json`、入口页面和 `README`。不要把整个样例目录塞入 AI 上下文。
