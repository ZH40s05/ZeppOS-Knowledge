# 自有知识

自有知识是由 ZHAO 维护、非官方但可分享的 ZeppOS 中文开发知识。它用于补足官方文档没有覆盖、没有讲清楚或与真机行为存在差异的部分。

## 内容边界

| 文件 | 内容 |
|------|------|
| `runtime-quirks.md` | 真机 runtime、UI、widget、framework 行为差异 |
| `re-summary.md` | 面向普通开发者的 RE 结论摘要 |
| `libraries.md` | 官方推荐库、社区库、自维护库的使用经验 |
| `project-reuse.md` | 跨项目复用结论和引用方式 |

## 写作要求

- 全中文，路径、API、命令和包名保持原文。
- 每条结论必须标注结论类型：`official`、`true-device confirmed`、`RE confirmed`、`inferred` 或 `needs test`。
- 说明适用范围、已验证设备/固件、证据来源、推荐做法和限制。
- 不粘贴大段官方文档，不复制固件、密钥、账号或凭据。
